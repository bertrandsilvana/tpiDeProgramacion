Juego de Anagramas de Animales

Este proyecto es un juego simple en JavaScript donde el usuario debe resolver un anagrama generado a partir de nombres de animales. El sistema selecciona una palabra al azar, mezcla sus letras y muestra el anagrama por consola.
El jugador tiene 3 intentos para adivinar la palabra correcta mediante entradas solicitadas con prompt().

Funcionamiento general

Selección aleatoria de un animal desde una matriz.

Mezcla aleatoria de las letras.

Interacción mediante consola y ventanas de entrada.

Control de hasta tres intentos.

Mensajes finales de acierto o error.

Ejecución

Abrir el archivo HTML que incluya el script.

Ejecutarlo en el navegador.

Consultar la consola para ver el anagrama generado.
